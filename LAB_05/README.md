# LAB 05 - PHP Session & Cookie (Shop Demo)

## Chức năng
- Login (POST) -> tạo session
- Remember username (cookie 7 ngày)
- Logout (POST + CSRF) -> xóa session + xóa cookie
- Bảo vệ trang: dashboard/products/cart
- Cart lưu trong session: add/update/remove/clear
